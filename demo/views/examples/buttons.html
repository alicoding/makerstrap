You should create buttons with the <button> tag plus one of the colour-specific classes:

class="btn btn-default"
class="btn btn-primary"
class="btn btn-secondary"
class="btn btn-info"
class="btn btn-warning"
class="btn btn-danger"

If you want to create buttons that are 100% width on mobile screens, use the class btn-block-mobile.
